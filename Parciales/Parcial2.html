Ejercicio 2

Utilizando el mtodo de inversin vamos a generar n=1000 valores de una distribucin Pareto con parmetros a=5 y b=4.

n<-1000
a<-5
b<-4

set.seed(1)
u<-runif(n)

Evaluamos usando grficos y el contraste de Kolmogorov-Smirnow que en efecto los valores generados provienen de dicha distribucin.

inversa.pareto<-function(u, a, b) {
  (b/(1-u)^(1/a))
}

x<-inversa.pareto(u,a,b)

Calculamos la funciones de densidad y distribucin

f.densidad<-function(x, a, b) {
  ifelse(x>=b, (a*b^a)/(x^(a+1)), 0)
}

f.distribucion<-function(x, a, b) {
  ifelse(x>=b, 1-(b/x)^a, 0)
}

Creamos el grfico, y comprobamos que los valores que hemos generado corresponde a los de la distribucin. Para ello vamos a representar los valores de la distribucin mediante un histograma, la densidad suavizada y la densidad real.

hist(x, breaks="FD", freq=FALSE, main="Mtodo de inversin", xlim=c(0,20),ylim=c(0,1.3))
lines(density(x), col="blue")
curve(f.densidad(x,a,b),add=TRUE,col=2)

Evaluamos el contraste de Kolmogorov-Smirnow

ks.test(x,f.distribucion,a=a,b=b)
## 
##  One-sample Kolmogorov-Smirnov test
## 
## data:  x
## D = 0.024366, p-value = 0.5928
## alternative hypothesis: two-sided

Podemos apreciar que el p-valor nos dice que no se aprecian desviaciones de la hiptesis nula planteada.
